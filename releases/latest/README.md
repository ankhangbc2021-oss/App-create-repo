# Bulk Repository Creator
**Version:** v1.1.0
**Build Date:** 2026-07-18

## Installation Instructions
1. Download the installer for your platform.
   - For a multi-language setup, use Bulk Repository Creator.exe.
   - For a standard Windows Installer, use Bulk Repository Creator.msi.
2. Run the downloaded file and follow the on-screen instructions.

## Update Instructions
The application includes a built-in auto-updater.
To check for updates manually, go to **Settings > System > Check for Updates** inside the app.

## GitHub Releases Usage
Upload the contents of this folder directly to your GitHub Releases page for v1.1.0.
Ensure that latest.json is configured to point to these artifacts so the in-app updater functions correctly.
