# Bulk Repository Creator v1.0.0
**Release Date:** 2026-06-28

## Improvements & Bug Fixes
* **Localization:** Comprehensive UI audit and Vietnamese translation support across all components.
* **Repositories Sync:** GitHub real-time synchronization, viewing, sorting, and manual refresh on the repos view.
* **History Page:** New advanced sorting, searching, filtering, and native default browser links for history cards.
* **Private Repositories:** Ensured proper configuration and validation for private repository visibility through Octokit.
* **UI/UX Polish:** Improved empty states, hover interactions, loading states, and responsive layouts.
* **Settings Persistance:** Synchronized all client settings directly with the background Node.js service via SQLite.
* **Updater:** Native Tauri automatic updates integrated.
