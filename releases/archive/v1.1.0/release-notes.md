# Bulk Repository Creation Tool v1.1.0
**Release Date:** 2026-07-18

## Improvements & Bug Fixes
* **Localization:** Comprehensive UI testing and Vietnamese translation support across all components.

* **Repository Synchronization:** Real-time GitHub synchronization, viewing, sorting, and manual refreshing on the repository view interface.

* **History Page:** New advanced sorting, searching, filtering functionality and native default browser links for history tabs.

* **Private Repositories:** Ensured proper configuration and authentication for private repository visibility via Octokit.

* **UI/User Experience Improvements:** Improved blank state, hover interaction, load state, and responsive layout.

* **Storage Settings:** Synchronizes all client settings directly with the background Node.js service via SQLite.

* **Memory Cleanup:** Added memory cleanup functionality.

* **Removes Unnecessary Folders:** Does not push folders like __pycache__, etc.

* **Appearance Update:** More colorful themes, but some functions are still not working.

* **AI Features:** Added AI integration for creating README.md, etc.

* **Updater:** Integrated native Tauri automatic updates.
